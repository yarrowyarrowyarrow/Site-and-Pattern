# Learning Progress

The master checklist for the curriculum in this folder. When a lesson's
"Done when" is true, change its `[ ]` to `[x]`, then watch the bars move:

```bash
python scripts/learning_progress.py
```

Commit your ticks as you go (`Learning: L1.3 complete`) — your git history
is the long-term record. A guard test keeps this list in sync with the
phase files, so a lesson can never silently vanish from either side.
